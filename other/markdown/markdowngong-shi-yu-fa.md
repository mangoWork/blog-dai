## 上下标 {#上下标}

* 上标 ^
* 下标 \_

| 名称 | 数学表达式 | markdown公式 |
| :--- | :--- | :--- |
| 上标 | abab | $a^b$ |
| 下标 | abab | $a\_b$ |

## 累加 {#累加}

* \sum\_{ }^{ }
* 累加号的上标下标的前后顺序可以互换。

| 名称 | 数学表达式 | markdown公式 |
| :--- | :--- | :--- |
| 求和号 | ∑3xn∑3xn | $\sum{3x^n}$ |
| 带范围求和 | ∑Nn=13xn∑n=1N3xn | $\sum\_{n=1}^N{3x^n}$ |
